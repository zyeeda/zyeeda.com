Zyeeda CDE
Introduction to Zyeeda Common Development Environment
Zyeeda CDE 是一套面向快速构建的企业级应用开发平台。在 Java 环境下，融合了众多优秀的
框架和组件，以良好的设计、统一的封装、精简的 API 及友好的界面，让开发人员能够以不可思
议的速度构建复杂的企业级应用程序。
